# Weather Forecasting Web Application

This project is a responsive and user-friendly Weather Forecasting Web Application developed using HTML5, CSS3, and JavaScript ES6+.

The application provides real-time weather information and a 5-day forecast for any searched city using the OpenWeatherMap API. It also includes weather alerts, dynamic weather icons, sunrise/sunset timings, and a clean responsive UI that works across desktop and mobile devices.

## Features
- Real-time weather updates
- 5-day weather forecast
- Sunrise and sunset timings
- Weather alert notifications
- Dynamic weather icons
- Responsive design
- Search weather by city name
- Error handling for invalid cities

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API
